ZeroChurn is built on the Google ADK 2.0 framework for the Agents for Business track. It orchestrates analysis, integrates custom database tools via MCP, protects customer data with a security gateway, and keeps a human in the loop for final approvals.

[1:00 - ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's walk through the architecture.
First, every query enters through the Security Checkpoint, which scrubs PII and checks for prompt injection.
If the request is clean, it flows to our Orchestrator Agent.
The orchestrator uses the Risk Analyzer sub-agent to query the customer database via our Model Context Protocol server.
If a customer is flagged as High risk, the Campaign Designer sub-agent is invoked to craft a personalized outreach email with a custom incentive.
The orchestrator then routes the draft to the Human Review node, which pauses execution to seek manual CSM approval before releasing the final results.
